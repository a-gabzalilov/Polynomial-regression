Параметры аргументной строки:
config_file.txt chunk_size
где config_file.txt - конфиг файл, а chunk_size - размер интервалов.

В конфиг файл записаны: 
- Input file
- Output file
- Log file

Структура Input file:
Первая строка - размер N
Вторая строка - через пробел записаны N элементов ряда t
Третья строка - через пробел записаны N элементов ряда Х(t)

Структура Output file:
Первая строка - интервал,
Затем "Degree" - степень полинома, "pValue" - значение pValue, затем через тире "true/false" - принятие/отказ гипотезы об однородности остатков,
"Sigma" - оценка среднеквадратичного отклонения, "betas" - оценка коэффициентов полинома, записанные через пробел.

Пример:
[0;1000]
Degree: 0 pValue: 0.60928 - true  Sigma:1.017 betas: 0.059
Degree: 1 ....

Также дополнительно генерируется файл "Leftovers.txt":
Записываются все остатки через пробел.
